Gestión Empresa

Una vez rellenados todos los campos de Empresa, Responsable y Tutor Laboral tendremos una serie de botones:


- Insertar: Con este botón insertaremos los datos a la tabla.

- Modificar: Con este botón podremos modificar los datos escritos anteriormente en los campos.

- Borrar: Con este botón eliminaremos los datos insertados en los campos.

- Ver tabla con registros: Con este botón podremos ver la tabla con los datos escritos anteriormente.
