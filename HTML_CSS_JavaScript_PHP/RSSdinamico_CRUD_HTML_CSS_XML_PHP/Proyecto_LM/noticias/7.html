Se filtra el videojuego Call of Duty Black Ops V

Una de las sagas más famosas de videojuegos

Ocio - Julio Morales - 27 ENERO 2020 - 18:29



La tienda virtual de videojuegos filtra por error el juego poniéndolo a la venta para todos los usuarios por error durante una hora, suficiente para que muchos modders descarguen el juego y puedan coger el código para realizar el 'crack' y liberarlo para todos los usuarios de PC de forma gratuita


La empresa 'Treyarch', ha decidio posponer el lanzamiento oficial del juego para poder modificarlo lo suficiente como para que merezca la pena obtener la nueva versión oficial en lugar de la filtrada.


En concreto, el juego nos da acceso a un modo campaña, battle royale, zombies y multijugador. No contaremos con pase de batalla


Por último la empresa ha decidio denunciar a la empresa 'Steam' por filtrar su producto y tendrán que pagar la misma cantidad recaudada del juego final cuando este se lance, que será el otra tienda virutal, llamada 'Epic Games Store'.
